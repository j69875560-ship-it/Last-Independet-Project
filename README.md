# Smart Facility Finder

## 📌 Overview
This web application helps users find and manage nearby facilities such as hospitals, restaurants, and gyms.

## 🚀 Features
- Add new facilities
- Store data using localStorage
- Sort facilities by rating or reviews
- Interactive map display
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Maps API

## 📂 Pages
- Home (index.html)
- Add Place (add.html)
- About (about.html)

## ⚠️ Known Issues
- Map requires API key
- Distance sorting not fully implemented

## ▶️ How to Run
1. Download project
2. Open in browser OR use Live Server
3. Add your Google Maps API key
