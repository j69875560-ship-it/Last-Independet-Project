Project Overview

Smart Facility Finder is a web-based application developed to help users easily discover nearby facilities such as cafés, hospitals, gyms, restaurants, schools, and other important locations.

The project was designed with the goal of improving accessibility and convenience when searching for facilities within a specific area. Many users often struggle to quickly identify nearby services, estimate travel distance, or determine the easiest route to reach a destination. This application addresses that challenge by combining interactive mapping, search functionality, routing, and dynamic facility management into a single easy-to-use platform.

The system allows users to search for facilities in real time, display them on an interactive map, calculate routes, estimate travel distance and travel time, and organize facilities based on different criteria such as ratings or reviews.

Key Features

  • 📍 Interactive map integration using OpenStreetMap and Leaflet
  • 🔍 Real-time facility searching and filtering
  • 🚗 Route generation and navigation between locations
  • 📏 Estimated travel distance in kilometers
  • ⏱ Estimated travel time calculations
  • 📂 Facility data storage using localStorage
  • 📱 Responsive user interface for desktop and mobile devices
  • ⭐ Facility sorting based on ratings and reviews
  • 📝 Ability for users to add custom facilities dynamically

Technologies Used

This project was built using core front-end web development technologies and modern JavaScript mapping libraries.

HTML5

Used to structure the web pages using semantic and accessible markup.

CSS3

Used for responsive layouts, styling, animations, and overall visual design.

JavaScript

Used for DOM manipulation, event handling, local storage functionality, and map interactions.

Leaflet & OpenStreetMap

Used to implement real interactive maps, routing, and nearby facility searches.
